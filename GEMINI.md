# JSON Type Definition (JTD) Spec Gemini Context

## Focus Area
**Deterministic Schema**

## Why Index?
Reference for strict, unambiguous data contracts used in regulatory audits.

## Project Description
JSON Type Definition (JTD) is a format for describing the shape of JSON messages. It is designed to be limited to the type systems of mainstream programming languages to enable code generation.

## Key Concepts
- **JTD**: JSON Type Definition (RFC 8927).
- **Goals**: Code generation, portable validation.
- **CDDL**: Concise Data Definition Language used for definition.

## Conventions
- **Language**: IETF Internet-Draft format.
- **Schema**: JTD format (Schemas, Definitions).
